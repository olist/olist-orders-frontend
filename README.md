# Next.js Boilerplate

Boilerplate project for front-end development and starting fast, made with Nextjs and Typescript.

##### Installation

```
  yarn install
```

##### Development Environment

```
  yarn dev
```

##### Product Environment

```
  yarn start
```

##### Build Artifact

```
  yarn build
```

##### Lint Code

```
  yarn lint
```

##### Launch Storybook

```
  yarn storybook
```
